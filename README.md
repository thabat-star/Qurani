# 🎧 قرآني - بديل إسلامي لتجربة الاستماع

---

## عن التطبيق
**قرآني** هو تطبيق استماع إسلامي مستوحى من فكرة تطبيق اسبوتفاي لكن بمحتوى نقي يقرّبك من الله. 

- 🎧 استماع للقرآن بأصوات قراء مختلفين
- 📚 مكتبة إسلامية (أناشيد / تلاوات / محتوى نافع)
- ⏯️ تشغيل في الخلفية
- 🌙 تصميم هادئ ومريح
